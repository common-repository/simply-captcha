A new captcha system that uses simple question based questions instead of the other complicated image based systems. It also uses a IP/Bot monitor that will block bots and IP addresses that are considered spammers due to hitting multiple forms or hitting the same form multiple times over and over. SimplyCaptcha has blocked 100% of spammers during testing. 

== Description ==

SimplyCaptcha is a new method of Captcha. It allows high customization, Simple questions are asked, and Users IP and User Agent is monitored for spam. It has a multi-layer security
monitoring system. It is constantly monitoring for spammers and spambots. If it finds one, it will block them instantly. Its cloud based, so if they are blocked on your site, they are block on everyone's site using SimplyCaptcha.

== Installation ==

e.g.

1. Expand zip document.
2. Drag and drop entire "simplycaptcha" plugin into your plugin directory.
3. Activate the plugin from the admin of your WordPress installation.
4. Once activated a tab will be created on the admin toolbar, it should be near the bottom, called "SimplyCaptcha." Click on that tab.
5. Place your key and your website url in there and turn on or off the areas you want captcha enabled or disabled. If you need a key, get it from http://www.simplycaptcha.com
6. Go to your website and check to make sure its working!

Comments, questions, see http://www.simplycaptcha.com
